# GO-Django
First Django project

# Requirements
- **Python > 2.x with pip** 
    - from page [www.python.org](https://www.python.org/downloads/) 

# Deployment guide

1.Activate virtual environment in main dir

```cmd
venv\Scripts\activate
```

2.Install modules

```cmd
pip install Django
```

3.Run application server

```cmd
python app\manage.py runserver
```
